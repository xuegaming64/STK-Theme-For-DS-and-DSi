This Supports theme for AOS, R4 Wood And TWilight Menu++!

For OG DS and DS Lite (AceCard AOS, R4 Wood):
1. Download My Zip File
2. Extract this Zip File
3. Open the Extracted Folder (like This: `C:\Users\YourUsername\Downloads\STK DS Skin Theme\STK DS Skin Theme`
4. Right Click and Copy This Folder of `STK DS Skin Theme` to  `_rpg / ui` for example `_rpg / ui / STK DS Skin Theme`
5. Now Paste And Insert Cardridge to DS Lite
6. Press START, Go to Settings And Choose User Interface Settings for themes and Select `STK DS Skin Theme`

For Dsi Twilight Menu++
1. Download my Zip File and Extract The Folder
2. Copy that Folder if you here: `C:\Users\YourUsername\Downloads\STK DS Skin Theme\STK DS Skin Theme <This Way and Use Right Click, Not instead Opening!!`
3. Paste into: `SdCard for dsi:\_nds\TWiLightMenu\akmenu\themes` Like This: `SdCard for dsi:\_nds\TWiLightMenu\akmenu\themes\STK DS Skin Theme`
4. Press START  to Into Classic DS Menu and Choose At the Bottom Center ds Icon (Settings)
5. Now Select In Twilight Menu Settings, Go To the Interface and Choose Wood*
6. After used Wood Choose Own Theme if there is a A Button, Select it and Set `STK DS Skin Theme` and Press a again
7. Now Press B to Take This Effect!!
